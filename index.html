Moderation

/clear - Delete messages

/ban - Ban a user

/unban - Unban a user

/kick - Kick a user

/lock - Lock a given channel

/unlock - Unlock a given channel

/mute - Mute a user from the server

/unmute - Unmute a user from the server

/add-role - Add role to a user

/remove-role - Remove role from a user

/play - Play song

/forward - Forward seconds in a song

/rewind - Rewind seconds in a song

/shuffle - Shuffle the songs i the queue

/loop - Set the loop of the song or the queue

/nowplaying - Display info about currently playing song
